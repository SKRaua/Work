定义 $f_i$ 为凑出 $i$ 元的纸币方式数，因此答案是 $f_w$。

需要初始化 $f_0 = 1$，因为凑出 0 元只有一种方式，就是不给钱。

凑出 $i$ 元的组合方式是把 $i$ 分别减去每种纸币面额后的方式数之和，即$\sum_{j=1}^{n} f_{\,i-a_j}.$

状态转移方程如下：

$$
f_i = \sum_{j=1}^n f_{i-a_j}
$$

注意取模。注意数组溢出。

⚠️ 它通过 **“最后一张用哪种纸币”** 的分类逻辑，将所有凑出 **i** 元的方式无重复、无遗漏地统计完全，且天然体现了 “顺序不同算不同方式” 的要求。任何一种凑出 **i** 元的支付序列， **最后一张纸币必然是 **n** 种面额中的某一种** （因为总共只有这 **n** 种纸币）。基于此，可以将所有方式按 “最后一张纸币的面额” 分成 **n** 个不重叠的类别：

```cpp
#include<iostream>
using namespace std;
int n,w,a[1005],f[10005];
const int mod=1e9+7;
int main(){
    cin>>n>>w;
    for(int i=1;i<=n;i++){
        cin>>a[i];
    }
    f[0]=1;
    for(int i=1;i<=w;i++){
        for(int j=1;j<=n;j++){
            if(i-a[j]>=0){
                f[i]=(f[i]+f[i-a[j]])%mod;
            }
        }
    }
    cout<<(f[w]%mod)<<endl;
    return 0;
}
```
